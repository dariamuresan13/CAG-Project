# Computer-Aided Graphics (CAG) Project

Academic project (Technical University of Cluj-Napoca) covering:
- AutoCAD technical drafting (2D)
- P&ID diagram drafting
- AutoCAD LISP macro ("Flower Power")
- C/OpenGL animated liquid flow visualization

## Structure
- `lisp/` – AutoCAD LISP macro
- `opengl/` – C/OpenGL visualization
- `docs/` – Full project report (PDF)
